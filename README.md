## Requirements

- **Install [NodeJS](https://nodejs.org/)**
- **Install [Expo](https://expo.io/)**
- **Create account [Yelp](https://www.yelp.com/)**

## Installation

```
cp .env.example .env
yarn install
```

## Env

 - Set **YELP_CLIENT_ID** and **YELP_API_KEY**

## Usage

```bash
yarn start
```
